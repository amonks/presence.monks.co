# presence.monks.co

the documentation website for `PRESENCE and the arrow of time`

more info on [the website itself](//presence.monks.co)

## develpoment

install [boot](https://github.com/boot-clj/boot), then run one of these commands:

* `boot dev -- run` to run a live-reloading local dev server with an unoptimized build
* `boot production -- run` to run a live-reloading local dev server with an unoptimized build
* `boot dev -- build` to build into `target/` an unoptimized build 
* `boot production -- build` to build into `target/` an optimized build
